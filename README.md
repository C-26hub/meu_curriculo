# 📄 Currículo em HTML e CSS

Este projeto é um currículo simples desenvolvido utilizando apenas **HTML5** e **CSS3**, com foco em estrutura semântica e estilização limpa.

## 🚀 Objetivo

O objetivo deste projeto é praticar conceitos básicos de desenvolvimento web, como:

- Estruturação de páginas com HTML
- Estilização com CSS
- Organização de conteúdo
- Criação de um layout responsivo simples

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JS

## 📁 Estrutura do projeto

```
📁meu_curriculo       
┣ 📄 README.md    
┗ 📁 curriculo      
    ┣ 📄 index.html      
    ┣ 📄 style.css     
    ┣ 📄 script.js     
    ┗ 📁 assets      
```

## 💻 Como visualizar o projeto

1. Clone este repositório: 

```bash
https://github.com/C-26hub/meu_curriculo.git` 
```

3. Acesse a pasta do projeto: curriculo

4. Abra o arquivo `index.html` no navegador.
